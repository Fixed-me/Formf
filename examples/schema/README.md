# Examples

This folder contains runnable examples for Formf.

## Available examples

- `schema/basic_schema_export.py`
  Small form and `to_schema()` output.
- `schema/advanced_schema_export.py`
  Field validators, form validators, and advanced `requiredif` schema output.

## Run

From repository root:

```bash
PYTHONPATH=src python3 Docs/examples/schema/basic_schema_export.py
PYTHONPATH=src python3 Docs/examples/schema/advanced_schema_export.py
```
